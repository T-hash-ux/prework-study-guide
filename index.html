HTML

  • HTML stands for Hyper Text Markup Language.
  • HTML elements work together to build the strure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.
  • HTML elements tell the browser how to display the content.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The "!DOCTYPE html" declaration tells the browser what document type it should expect when it is opened
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.

CSS

  • CSS (Cascading Style Sheets) is a stylesheet language used for describing the look and formatting of a document written in HTML.
  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a distributed version control system (VCS) used for tracking changes in source code during software development.
  • Git allows multiple developers to work on the same project simultaneously and keep track of changes made by each developer.
  • Git allows you to revert to previous versions of the code and collaborate with others on the same project.
  • Git has three main components: repository (a place to store all files and changes), working directory (a copy of the files on your local machine), and stage (a place to prepare changes for a commit).
  • Git commands: clone (create a local copy of a remote repository), add (add changes to the stage), commit (save changes to the repository), push (send changes to a remote repository), pull (receive changes from a remote repository), etc.
  • Git branching allows multiple developers to work on different features of the same project simultaneously. The master branch is the default branch, and new branches can be created for specific features.
  • Git merge is used to combine changes from different branches into a single branch. Git merge conflicts occur when the same lines of code have been changed in different branches.

JavaScript

  • JavaScript is a high-level, dynamic, and interpreted programming language.
  • It is primarily used for creating interactivity and dynamic effects on websites.
  • JavaScript has six primitive data types: Number, String, Boolean, Null, Undefined, and Symbol.
  • In JavaScript, to declare a variable, we use the var keyword. After the var keyword, we give the variable a name. This name must be unique because it is how we will reference the information stored in the variable in our code.
  • we use an assignment operator "=" to assign the specific data that we want stored in our variable.
